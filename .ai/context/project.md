Sistema de encurtador de links de alta performance e alta concorrência.

Usuários podem:
- Se registrar
- Autenticar
- Criar links encurtados
- Visualizar estatísticas de seus próprios links

O sistema deve:
- Redirecionar links extremamente rápido
- Registrar estatísticas de acesso sem bloquear o redirecionamento
- Ser preparado para alto tráfego e escala horizontal
